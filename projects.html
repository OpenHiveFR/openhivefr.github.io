Le serveur Discord unicaen

Il s'agit du premier projet d'OpenHive. Un serveur de discussion discord dédié aux étudiants en informatique de l'université de Caen.

Le serveur possède actuellement une très bonne base et ne nécessite plus que la mise en place d'outil d'administation ainsi que quelques outils d'optimisation (autoroles, etc).
Cependant, le serveur nécessite également un renouvellement régulier de l'équipe, afin de garantir un bon contact et une bonne représentation des étudiants.

La documentation de Discord est disponible ici. Téléchargez Discord ici

Pour rejoindre le serveur, cliquez ici.

Le serveur est voué à être étendu. Plusieurs pistes sont actuellement étudiées : ouverture vers d'autres établissements, création d'autres serveurs avec des modèles...

Le site Github Pages d'OpenHive

Le site Github Pages d'OpenHive a pour but d'être une vitrine pour le projet. Répertorisation des sous-projets, des membres...
Temporairement, jusqu'à ce qu'une meilleure solution soit trouvée, le site contiendra aussi des ressources de type cours, tutoriels et exercices.

Actuellement, le site en est au stade de fondation (rédaction et style).

Pour plus d'informations, reportez-vous à l'accueil du site.

Une fois rédigé, le site sera mis à jour régulièrement.

RayBorn, un moteur grapique 3D

RayBorn est un projet de moteur 3D par RayTracing en C++.
Ce moteur n'a pas pour but premier d'être optimisé ni commercial (un projet de moteur 3D libre de qualité commerciale sera cependant envisagé plus tard) mais éducatif. Il sera très documenté, commenté, et son code sera aussi clair que possible tout en respectant au mieux les conventions de codage et les bonnes pratiques.

Le projet est actuellement non-fonctionel, en état de développement. Son code est disponible sur github.

Pour le moment, le projet est visible sur son repo github.

La documentation est pour l'instant uniquement constituée de commentaires présents sur le code.

Une fois rendu fonctionel, la phase d'optimisation démarrera.
